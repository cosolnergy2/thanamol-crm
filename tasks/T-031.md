# T-031 — E2E Tests — Contract Management (T-028, T-029, T-030)

**Status:** Test Pass
**Assignee:** tester

## Objective

Write Playwright E2E tests covering the Contract Management features implemented in T-028 (contract API), T-029 (contract pages), and T-030 (lease agreement pages).

## Definition of Done

1. Contract list page renders with stats cards (Total, Active, Pending Approval, Approved) and table headers (Contract No., Type, Status, Value).
2. Action buttons "New Contract" and "Expiring" are visible on the contracts list page.
3. Status and type filter dropdowns on the contracts list work correctly.
4. Create contract form renders with required fields (Customer, Project, Contract Type, Start Date).
5. New contract can be created and redirects to the contracts list.
6. Validation errors show when required fields are missing on create form.
7. Contract detail page shows all fields (Start Date, End Date, Status, Type, Contract Information, Customer, Project, Contract Value).
8. DRAFT contract can be edited and saved via the edit form.
9. DRAFT contract can be submitted for approval (status changes to Pending Approval).
10. PENDING_APPROVAL contract can be approved (status changes to Approved).
11. PENDING_APPROVAL contract can be rejected with a reason.
12. Reject dialog shows error when submitted without a reason.
13. Expiring Contracts page renders with urgency stats (Critical, Warning, Notice, Total Expiring).
14. Navigation from contracts list to expiring contracts via the Expiring button works.
15. Expiring contracts table headers are visible.
16. Contract print page renders with CONTRACT heading and contract data.
17. Lease Agreement list page renders with stats cards and New Lease Agreement button.
18. Lease Agreement status filter works.
19. New Lease Agreement form renders with Contract selector, Status, Lease Terms, Special Conditions.
20. New lease agreement can be created with a contract reference and redirects to list.
21. Lease Agreement create form validates that Contract selection is required.
22. Lease Agreement detail page shows contract reference fields.
23. DRAFT lease agreement can be edited and saved.

## Test Results

**Run command:** `pnpm --filter web exec playwright test e2e/T-031-contract-management.spec.ts --config=playwright.e2e.config.ts`

**Result: 23 passed, 1 skipped, 0 failed**
