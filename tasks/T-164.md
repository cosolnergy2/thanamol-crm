# T-164: Stock Issue + GRN form enhancement

**Status:** Ready to Dev
**Assignee:** dev
**Created:** 2026-04-01
**Updated:** 2026-04-01

### Objective

Enhance both the Stock Issue and Goods Received Note (GRN) forms to expose the new schema fields added in T-153. Stock Issue needs an auto-generated Issue Number and a Purpose dropdown. GRN needs a PO selection that auto-populates line items and a QC Status dropdown. Both forms must update inventory stock levels correctly on save.

Acceptance criteria:
- **Stock Issue:** API create/update schemas accept `purpose`; Issue Number auto-generates (read-only); Purpose dropdown sourced from `STOCK_ISSUE_PURPOSES` constant; item list is filtered by selected site
- **GRN:** API create/update schemas accept `po_id` and `qc_status`; PO selection dropdown auto-populates received items from the PO's line items; QC Status dropdown sourced from `QC_STATUSES` constant (Pass/Fail/Partial)
- Accepting a GRN increments inventory current_quantity for each received item
- Creating a Stock Issue decrements inventory current_quantity for each issued item
- All dropdown values sourced from constants in `packages/shared` — no hardcoded strings
