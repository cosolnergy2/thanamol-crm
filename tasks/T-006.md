- Replaced `next-themes` dependency in `sonner.tsx` with a static `theme="system"` prop (no Next.js in this project).
- Created barrel `index.ts` exporting all components (aliased sonner's `Toaster` as `SonnerToaster` to avoid name clash with toast `Toaster`).
