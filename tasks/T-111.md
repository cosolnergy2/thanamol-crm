### Objective
Create Zone CRUD API routes and shared TypeScript types for FMS. This is Phase 0 of the Facility Management System and provides the API layer that the frontend (T-112) depends on.

Acceptance criteria:
- Zone CRUD API endpoints respond correctly
- Shared types exported and usable from packages/shared
- `pnpm build` passes
- `pnpm test` passes

**API endpoints:**
- `GET /api/fms/zones?projectId=` — list with pagination, optional parentZoneId/search
- `GET /api/fms/zones/:id` — get zone with children and parent
- `POST /api/fms/zones` — create zone (409 on duplicate code per project)
- `PUT /api/fms/zones/:id` — update zone
- `DELETE /api/fms/zones/:id` — delete zone (409 if has children or units)
