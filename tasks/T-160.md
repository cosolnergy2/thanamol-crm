**Status:** Ready to Dev
**Assignee:** dev
**Created:** 2026-04-01
**Updated:** 2026-04-01

### Objective

Enhance the Preventive Maintenance Schedule API and create/detail forms to expose all new schema fields added in T-152. The current form is missing Scope Type, Trigger Type, Estimated Duration, a dynamic Maintenance Checklist/Tasks editor, Spare Parts selection from inventory, and Auto WO Generation toggle.

Acceptance criteria:
- API create and update schemas accept `scope_type`, `trigger_type`, `estimated_duration`, `spare_parts` (Json), `auto_create_wo`, `auto_wo_days_before`
- Create form displays PM Code (auto-generated, read-only), Scope Type dropdown, Trigger Type dropdown, Frequency Type dropdown
- Maintenance Checklist editor: dynamic rows with task name, description, duration
- Spare Parts section: select items from inventory
- Auto WO Generation toggle: when ON, reveal a days-before numeric input
- Detail page renders checklist and spare parts
- All dropdown values sourced from shared constants
