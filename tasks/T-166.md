# T-166: Inventory & Stock detail pages

**Status:** Ready to Dev
**Assignee:** dev
**Created:** 2026-04-01
**Updated:** 2026-04-01

### Objective

Create or enhance the detail/edit pages for Inventory Items, Stock Issues, and GRNs so that all fields — including those added in T-153 and the forms built in T-163 and T-164 — are visible and editable after creation.

Acceptance criteria:
- Inventory item detail page displays all fields: Item Code, Type, Barcode, Category, Unit, Company/Site, Vendor, Lead Time, Specifications, stock levels (current/reorder/min/max), unit price, photos
- Inventory item edit page pre-fills all fields and allows saving changes
- Stock Issue detail page shows header (Issue Number, Date, Purpose, Site, Created By) and a line-items table (item name, quantity, unit)
- GRN detail page shows header (GRN Number, Date, PO reference, QC Status, Received By) and a line-items table (item name, ordered qty, received qty, unit price)
- All detail pages are read-only except for the inventory edit page
