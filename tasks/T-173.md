# T-173: Vendor Contract create/edit forms

**Status:** Ready to Dev
**Assignee:** dev
**Created:** 2026-04-01
**Updated:** 2026-04-01

### Objective
Build dedicated create and edit pages for vendor contracts. The existing vendor contract flow is either missing or using a basic inline dialog. A full-page form is needed to capture contract details, SLA commitments, and rate cards.

Acceptance criteria:
- Contract Number auto-generates (read-only)
- Vendor selection dropdown links to existing vendor list
- Contract Type dropdown: Service Agreement / Maintenance / Supply / Rental / Consulting
- Service Category field (text or dropdown)
- Site selector, Start Date, End Date, Contract Value fields
- SLA section: dynamic rows with columns — Service Type, Response Time (hrs), Resolution Time (hrs), Penalty
- Rate Card section: dynamic rows with columns — Service Description, Unit, Rate (THB)
- Settings: Alert Days Before Expiry (number input), Auto-Renew toggle
- Submitting the form creates the contract with Draft status and it appears in the vendor's Contracts tab
- Contract detail page displays SLA and Rate Card as formatted tables
