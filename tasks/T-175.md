# T-175: Vendor Quotation fix + comparison page

**Status:** Ready to Test
**Assignee:** dev
**Created:** 2026-04-01
**Updated:** 2026-04-01

### Objective
Fix the broken navigation for Vendor Quotations (currently the sidebar link routes to vendor registration instead of the quotations page) and build a quotation comparison view that allows procurement staff to evaluate multiple vendor quotes against a single PR side by side.

Acceptance criteria:
- Sidebar "Quotations" link navigates to the vendor quotations list page, not to vendor registration
- A quotation can be created and linked to an existing PR
- The quotation comparison page for a PR shows all vendor quotes submitted for that PR in a side-by-side table
- The comparison table includes at minimum: vendor name, unit price, total price, delivery terms, notes
- Lowest price is visually highlighted in the comparison table

### Detail
**Implementation approach:**
- Sidebar path for \"Vendor Quotations\" was already correct. Removed 3 stale duplicate disabled entries (Assets, Work Orders, Inventory) that caused React key conflicts.
- Added \"Quotation Compare\" entry to sidebar at `/facility-management/procurement/quotations/compare`.
- Created new comparison page `compare.tsx` with side-by-side table: accepts `?prId` search param, shows all quotations for selected PR, highlights cheapest total and lowest unit price per item in teal, and allows selecting a quotation.
- Enhanced `index.tsx` to add a header \"Compare\" button and per-PR-group compare shortcut button when 2+ quotations exist.
- Exported `lowestTotalId` helper for unit testing.
