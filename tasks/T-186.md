### API Endpoints
- `GET /api/fms/landscape` — list landscape tasks (query: projectId, status, zoneId, page, limit)
- `GET /api/fms/landscape/:id` — get single task
- `POST /api/fms/landscape` — create task
- `PUT /api/fms/landscape/:id` — update task
- `DELETE /api/fms/landscape/:id` — delete task
- `GET /api/fms/waste` — list waste records (query: projectId, wasteType, page, limit)
- `GET /api/fms/waste/:id` — get single record
- `POST /api/fms/waste` — create record
- `PUT /api/fms/waste/:id` — update record
- `DELETE /api/fms/waste/:id` — delete record

### Database Model Changes
- `LandscapeTask`: title, description, project_id, zone_id, scheduled_date, completed_date, status (SCHEDULED/IN_PROGRESS/COMPLETED), assigned_to, notes
- `WasteRecord`: record_date, project_id, waste_type (GENERAL/RECYCLABLE/HAZARDOUS/ORGANIC), volume (Float), unit (kg/ton/m3), disposal_method, vendor_id, cost, notes
